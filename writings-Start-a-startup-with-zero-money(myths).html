Start a startup with zero money(myths)

7 June 2022

If you search the term how to start a startup with zero money. You will find several websites and techniques on the internet. For the past week, one of my friends asking me this same question, do I need money to start a startup, I have many ideas? He has already watched several videos on youtube. Many gurus taught this how to start a startup without money. I don't know why people search for these things on the internet in the first place - it is very simple. If you want to start a startup, calculate the expenses and go for it.

I know one thing the best time is to start a startup is when you are in college. You can begin as a side project. When I was in college, I didn't know about a startup. But I was always interested in projects. You can say Impact-based social projects. During that time the best project on which I had worked was Prixfit. I built Prixfit to help farmers. I will talk about Prixfit in some other posts. But that college project also cost me some thousand money.

Some people say to start a podcast with zero money. But podcasts also need some setups and equipment, like a mic, laptop, internet, and many more then how can someone start any startup for free?
